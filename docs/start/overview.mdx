# Ante

Ante is a lightweight AI agent that lives in your terminal, built by [Antigma Labs](https://antigma.ai). It focuses on coding and daily engineering tasks, designed from the ground up in native Rust for security, performance, and resistance to bloat.

<Warning>
  Ante is currently in beta and provided as a research preview. It may contain experimental features or incomplete functionality. Use with caution in production environments.
</Warning>

## Why Ante?

- **Native Rust** вҖ” Fast startup, low memory footprint, no runtime dependencies
- **Terminal-first** вҖ” Rich TUI for interactive sessions, headless mode for scripting and CI
- **Provider agnostic** вҖ” Works with Anthropic, OpenAI, Gemini, Grok, Open Router, and local models
- **Offline capable** вҖ” Run models locally via llama.cpp with no API keys required
- **Extensible** вҖ” [Agent Skills](https://agentskills.io), sub-agents
- **Secure by default** вҖ” Tool approval gates, configurable permissions, no ambient authority

## How it works

Ante uses a client-daemon architecture. The TUI (or headless runner) is the client that sends operations to a daemon. The daemon manages sessions, communicates with LLM providers, and executes tools on your behalf.

в”Ӯ  Client  в”Ӯ в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–¶ в”Ӯ  Daemon  в”Ӯ в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–¶ в”Ӯ Provider в”Ӯ
в”Ӯ (TUI/CLI)в”Ӯ в—Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ в”Ӯ (Session)в”Ӯ в—Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ в”Ӯ  (LLM)   в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ     Evt      в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ              в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

The client sends `Op` (operations) like user input, and the daemon responds with `Evt` (events) like agent messages and tool calls.
